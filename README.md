# 📚 Knowledge Base

自分のアウトプットを蓄積・整理するリポジトリ。
書籍・ブログ執筆の素材置き場でもある。

## 📁 構成

| フォルダ | 内容 | 使い方 |
|---------|------|--------|
| [`daily/`](./daily/) | 日々の考え・メモ | 日付ファイルに雑多に投げ込む |
| [`tech/`](./tech/) | 技術的な知見 | カテゴリ別に整理 |
| [`books/`](./books/) | 読書メモ・書評 | 1冊1ファイル |
| [`drafts/`](./drafts/) | ブログ・記事の下書き | wip→published に昇格 |

## ✍️ 書き方のルール

各ファイルの冒頭に以下を書く（最小限でOK）：
```
---
date: YYYY-MM-DD
tags: [タグ1, タグ2]
status: draft | published
---
```

## 🗺️ 使い方フロー
```
思いついた → daily/ に投げる
↓
ある程度溜まった → テーマで GitHub Issue を立てる
↓
Issue を元に drafts/wip/ で記事化
↓
完成 → drafts/published/ に移動 & 外部公開
```
